# Key Forge — Gerador de Senhas

O Key Forge é um gerador de senhas feito em Python com interface gráfica usando Tkinter.

Ele permite criar senhas seguras e personalizadas de forma rápida, direto pela interface.


# Sobre o projeto

Este projeto foi desenvolvido com foco em prática e aprendizado de:

- Python na prática
- Interfaces gráficas com Tkinter
- Uso do módulo `secrets` para geração segura de senhas
- Organização básica de código em módulos


# Funcionalidades

- Geração de senhas seguras usando `secrets`
- Definição do tamanho da senha (6 a 64 caracteres)
- Opção de incluir números
- Opção de incluir caracteres especiais
- Botão para copiar a senha gerada


# Tecnologias utilizadas

- Python
- Tkinter
- `secrets` (geração criptograficamente segura)
- `string` (conjuntos de caracteres)
